Production Line Application

Production Line/ Worker Line Application :- It is an application which is used by production line workers for problems creation on manufacturing line. This application contains various modules for worker, supplier, admin etc.


Tech Stack : Java, Angular, Framework : Spring Boot,Tools : IntelliJ IDEA, DataGrip, MySQL, MacPass, Vulnerabilities : BlackDuck Scan, Cloud : Openshift, CI/CD : Bamboo and Argo CD


  1. This application is used by the production line workers in order to raise any complaints, parts information, faulty parts, Order information with client
  2. Used Java, Spring Boot, JPA, Spring Security, Junit Mockito for unit test cases, Profiling etc
  3. Data for this application is stored in MySQL Database and being rendered over UI which is created with the help of Angular
  4. Used Blackduck for code related Vulnerability Scans, Bamboo for Continious Integration and ArgoCD for Continous Deployment
  5. Used CRON Jobs for automated tasks to be triggered over specific period of time.
  6. This application enhances the problem management cycle for workers over production line and makes the tracking of the problem status easily

WEB Based Portal

Application for Copying up of data from one location to another and used by other interfaces :- It is an application which is used by production line workers for problems creation on manufacturing line. This application contains various modules for worker, supplier, admin etc.


Tech Stack : Java, CSS and Tailwind CSS Framework : Servlet,Tools : Eclipse, Mysql Workbench, Power BI Desktop


  1. This application was created to transfer data from one location to another location and then process its data and used by further interfaces.
  2. Used Java for backend and servlets as a framework, HTML and CSS for frontend and mysql as a database.
  3. Used by Scientist to have there data protected in other location.
  4. Created JOBS for copying files form one location to another.
  5. These files then used by other systems which consist of Power Bi for reports and Power apps for apps.
  6. Files copied will then removed the original location after specific time.
  7. Used Power BI to analyse the data and then process it on power bi dashboards.
